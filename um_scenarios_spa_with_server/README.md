Запуск проекта (встроенный мини-сервер)
======================================

Вариант A. Python (без зависимостей)
------------------------------------
1) Откройте терминал в папке проекта
2) Запустите:
   python serve.py
3) Откройте в браузере:
   http://127.0.0.1:8000

Вариант B. Node.js + Express
----------------------------
1) Установите зависимости:
   npm install
2) Запустите сервер:
   npm start
3) Откройте:
   http://127.0.0.1:8000

Примечания
----------
- Файл data.json отдаётся как статический ресурс с того же origin — fetch работает без CORS-проблем.
- Основной вход: index.html, стили и скрипт минифицированы (app.min.css, app.min.js).
- Для Lighthouse-замеров используйте режимы Mobile и Desktop и сохраните отчёты в папку /audit вашего репозитория.
